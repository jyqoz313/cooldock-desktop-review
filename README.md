## 酷呆桌面是什么？一句话说清楚

它是北京酷呆科技做的一款Windows桌面整理工具，干的事和Stardock Fences很像，但风格更轻、更安静。装上之后，你可以在桌面任意位置拉出一个个半透明的"盒子"分区，把快捷方式、文件夹、图片、文档往里一拖，桌面瞬间清爽。

光手动拖还不算什么。真正好用的是它能**设定规则自动归类**——比如你规定"凡是.mp3自动进音乐盒子""截图工具存下来的图片自动进素材盒子"，新文件一落地就被归好位，你啥都不用管。

更妙的是它支持**文件夹映射**：把某个真实文件夹映射成桌面上的一个盒子，直接在桌面浏览和操作里面的文件，不用再一层层点进去。我个人的工作流是把"本周项目"文件夹映射到桌面，每天一开电脑就能看到要处理的活儿。

还有个我特别喜欢的功能叫**盒子置顶**——按快捷键把某个盒子顶到最前面，处理完再收回去。比如剪辑视频时把"素材盒子"置顶，要什么拖什么，不用切回桌面再切回来。
